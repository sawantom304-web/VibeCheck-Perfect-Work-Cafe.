# VibeCheck Cafe Finder

This project is a web application for finding and booking workspaces in cafes based on their vibe, WiFi speed, and amenities.

## Files Summary

- **cafe.html**: The main landing page featuring a hero section, cafe search by vibe (quiet, social, busy), cafe cards with images and details, an embedded Google Maps section for nearby cafes in Panvel, customer reviews, and an about section.

- **availability.html**: A desk booking page with 6 clickable desks that toggle between available (green) and booked (red) states. Includes a back button to return to the main cafe page.

- **script.js**: Contains JavaScript functions for desk booking (`toggleDesk`) and cafe filtering (`filterCafes`) based on selected vibe.

- **style.css**: Provides styling for a dark theme with a coffee background image, animations (e.g., steam effect), responsive layouts, and hover effects for interactive elements.
